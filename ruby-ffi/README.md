# Cobhan

## Usage

### Running specs

To run specs across all distros

    ./bin/test.sh

To run tests for specific distro

    ./bin/test.sh alpine

### Building gem

To build gems across all distros

    ./bin/build.sh

To build a gem for specific distro

    ./bin/build.sh alpine

### Starting shell

To start a shell for specific distro

    ./bin/shell.sh alpine
